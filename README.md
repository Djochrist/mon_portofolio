# Portfolio — Djochrist K.

Portfolio personnel orienté `Data` et `Web`, construit avec `Vite`, `React`, `Tailwind CSS` et `Framer Motion`.

## Développement

```bash
npm install
npm run dev:client
```

## Build

```bash
npm run build:client
```

Le build statique est généré dans `dist/public`.

## Stack

- `React`
- `TypeScript`
- `Tailwind CSS`
- `Framer Motion`
- `Vite`
